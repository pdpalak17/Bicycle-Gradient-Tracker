# Bicycle Gradient Tracker 

A Computer Vision based system that detects road gradient and calculates required pedal force using an Intel RealSense T265 camera and OpenCV.

## Features

- Real-time horizon detection using OpenCV
- Road gradient estimation from camera pitch
- Pedal force estimation for uphill cycling
- Exponential Moving Average (EMA) smoothing
- CSV data logging for ride analysis
- Real-time visual display

## Technologies Used

- Python
- OpenCV
- Intel RealSense SDK
- NumPy

## Hardware Requirements

- Intel RealSense T265 camera
- Linux / Raspberry Pi system
- Bicycle mount for camera
